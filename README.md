# 修改圖片檔名前墜工具

---

## 使用方法 Get started

### Step.1 建立 .env 環境變數檔案

內容如下

```shell
USER_NAME = "<Your Name>"
FOLDER_NAME = "<Folder Name>"
```

### Step.2 更改目標資料夾與使用者前墜

```shell
USER_NAME = "FKT" # 輸出格式為 FKT-0001.jpg
FOLDER_NAME = "./" # 以當前資料夾所有圖片進行轉換
```

### Step.3 運行程式

```shell
python main.py
```
